# UI_SPEC — 界面规格说明

## 1. 整体布局

三栏横向布局，从左到右依次为：导航栏、左侧面板、右侧面板。

```
┌──────┬──────────────────────┬──────────────────────────────────────────────┐
│ Nav  │    Left Panel        │              Right Panel                     │
│ 48px │      484px           │                 1354px (flex: 1)            │
└──────┴──────────────────────┴──────────────────────────────────────────────┘
```

- **导航栏**：`width: 48px; flex-shrink: 0`
- **左侧面板**：`width: 484px; flex-shrink: 0`
- **右侧面板**：`flex: 1`（最小参考宽度 1354px）
- 整体参考总宽：1886px，高度 100vh

---

## 2. UI 组件库 & 设计风格

- **UI 组件库**：Ant Design 5
- **主题模式**：**Dark 暗色主题**，全局启用 `theme.darkAlgorithm`
- **主题色**：`#1677FF`（Ant Design 默认蓝，暗色下对比度良好）
- **背景色**：
  - 导航栏：`#0A0F14`
  - 左侧面板：`#0D1117`
  - 右侧面板：`#0D1117`
  - 卡片/容器背景：`#161B22`
  - 输入框背景：`#1C2128`
- **整体风格**：深色科技感 / 网络监控风
- **语言**：中文
- **字体**：系统默认，代码区使用等宽字体

---

## 3. 导航栏（48px 宽，全高）

### 3.1 布局描述

固定在最左侧，全屏高度。顶部放 Logo，下方竖排图标按钮，底部可放设置等次要图标。

### 3.2 组件清单

| 组件 | 类型 | 位置 | 说明 |
|------|------|------|------|
| Logo | SVG 图标 | 顶部居中 | 品牌标识，不可点击或点击回首页 |
| 会话列表按钮 | SVG 图标按钮 | 顶部 Logo 下方 | hover 显示 tooltip「会话列表」，点击切换左侧为列表视图 |
| 设置按钮（预留） | SVG 图标按钮 | 底部 | hover 显示 tooltip「设置」 |

### 3.3 交互行为

- 所有图标按钮：hover 时在右侧显示 tooltip 文字（Ant Design Tooltip 组件，placement="right"）
- 当前激活的视图对应图标高亮显示
- 点击「会话列表」图标 → 左侧面板切换为会话列表视图

---

## 4. 左侧面板 — 视图一：会话列表（484px）

### 4.1 布局描述

顶部标题行（标题 + 新建按钮） + 下方可滚动会话列表。

### 4.2 组件清单

| 组件 | 类型 | 位置 | 说明 |
|------|------|------|------|
| 标题「会话列表」 | 文本 | 顶部左侧 | 固定标题 |
| 新建对话按钮 | 按钮（图标+文字） | 顶部右侧 | 点击创建新会话并进入对话界面 |
| 会话列表 | 可滚动列表 | 标题下方 | 每条为一个会话卡片 |

### 4.3 会话列表项

每条会话卡片展示：
- 会话标题（第一行，加粗）
- 最后消息时间（右上角，灰色小字）
- 消息内容摘要（第二行，截断省略）

### 4.4 交互行为

- 点击任意会话卡片 → 左侧切换为对话界面，加载该会话历史消息
- 点击「新建对话」→ 创建新会话，左侧切换为对话界面（消息区为空）
- hover 会话卡片 → 背景高亮，显示删除图标

## 5. 左侧面板 — 视图二：对话界面（484px）

### 5.1 布局描述

顶部固定导航栏 + 中间可滚动消息区 + 底部悬浮输入气泡。

```
┌─────────────────────────────┐
│ ← [会话标题]                 │  ← 顶部栏，固定
├─────────────────────────────┤
│                             │
│   [用户提问气泡]             │
│                             │
│   [步骤卡片 1]              │
│   [步骤卡片 2]              │
│                             │
│   [结论容器]                │
│                             │
│         ↕ 可滚动            │
├─────────────────────────────┤
│   [底部输入气泡 悬浮]        │  ← 底部固定
└─────────────────────────────┘
```

### 5.2 组件清单

| 组件 | 类型 | 位置 | 说明 |
|------|------|------|------|
| 返回按钮 + 标题 | 图标 + 文本 | 顶部栏 | ⬅️ 点击返回会话列表 |
| 消息滚动区 | 可滚动容器 | 中间 | 包含所有消息组件 |
| 用户提问气泡 | 文本卡片 | 消息区 | 用户每次提问展示 |
| 步骤卡片 | 可折叠卡片 | 消息区 | Agent 执行步骤，见 5.3 |
| 结论容器 | 复合组件 | 消息区 | 思考过程 + 最终结论，见 5.4 |
| 底部输入气泡 | 输入框容器 | 底部悬浮 | 见 5.5 |

### 5.3 步骤卡片规格

- **折叠态**：`width: 452px; height: 56px`，圆角 `border-radius: 28px`（椭圆切角），左侧显示步骤编号/图标，中间显示步骤标题，右侧显示展开/收起箭头
- **展开态**：高度自适应，内部显示子步骤时间轴

子步骤时间轴每条包含：
- 左侧竖线 + 节点圆点
- 子步骤名称（加粗）
- 子步骤结果（浅色文本）
- 右侧：完成时间（如 `12:34:56` 或耗时 `2.3s`）

### 5.4 结论容器规格

```
┌─────────────────────────────────────┐
│ ✅ 已深度思考（12s）  [▼展开思考]   │  ← 可点击展开/收起
│ ─────────────────────────────────── │
│ [思考过程文本，展开后可见]           │  ← 可折叠区域
├─────────────────────────────────────┤
│ [最终结论文本，支持 Markdown 渲染]   │  ← 不可折叠，常驻
└─────────────────────────────────────┘
```

- 思考过程区域：灰色背景，斜体文本，支持折叠
- 结论区域：正常文本，支持 Markdown（加粗、列表、代码块等）

### 5.5 底部输入气泡规格

- **尺寸**：`width: 452px; height: 96px`，圆角 `border-radius: 24px`
- **内部布局**：上方为文本输入区（多行，可扩展），下方为操作栏
- **深度思考按钮**：`width: 88px; height: 28px`，圆角，位于操作栏左侧，点击切换激活态（纯 UI，不与后端交互）
- **发送按钮**：位于操作栏右侧，图标按钮，回车或点击触发发送

### 5.6 消息布局顺序（每轮对话）

```
[用户提问气泡]          ← 用户发送后立即出现
[步骤卡片 1]            ← Agent 执行时流式出现
[步骤卡片 2]
[步骤卡片 N]
[结论容器]              ← 最后出现，包含思考过程和最终结论
```

## 6. 右侧展示面板（1354px，flex:1）

### 6.1 布局描述

全高展示区，根据 Agent 返回的渲染指令动态切换内容类型。无内容时展示空状态。

### 6.2 空状态

居中显示提示图标 + 文案「等待 Agent 输出结果...」

### 6.3 场景一：图像展示

触发条件：Agent 返回 `type: "image"` 的渲染块

```
┌──────────────────────────────────────────────────┐
│                  [标题文本]                       │
│  ┌────────────────────────────────────────────┐  │
│  │                                            │  │
│  │              大图展示区                    │  │
│  │         (max-height: 70vh, 居中)           │  │
│  │                                            │  │
│  └────────────────────────────────────────────┘  │
│                  [结论文本]                       │
│           (支持 Markdown 渲染)                   │
└──────────────────────────────────────────────────┘
```

- 图片宽度铺满右侧内容区，高度等比缩放
- 支持点击图片全屏预览

### 6.4 场景二：数据洞察

触发条件：Agent 返回 `type: "insight"` 的渲染块

```
┌──────────────────────────────────────────────────┐
│  [图表 1]              │  [图表 2]               │
│  ECharts 渲染          │  ECharts 渲染            │
│  [图表 1 文字结论]     │  [图表 2 文字结论]       │
├──────────────────────────────────────────────────┤
│  [图表 3]              │  [图表 4]               │
├──────────────────────────────────────────────────┤
│  📄 完整 Markdown 报告（可滚动）                  │
└──────────────────────────────────────────────────┘
```

- 图表区：2列网格布局，每个图表卡片包含 ECharts 图 + 下方文字结论
- Markdown 报告区：位于图表下方，全宽，支持完整 Markdown 渲染（标题、表格、代码块等）
